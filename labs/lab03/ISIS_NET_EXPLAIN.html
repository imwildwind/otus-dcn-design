AFI Всегда 49 — обозначает «частное» (private) адресное пространство IS-IS (аналог RFC 1918 для IP)
Area ID 1–13 байт, записывается группами по 2 байта. Все роутеры одной зоны имеют одинаковый Area ID. Минимум — 0001
System ID Ровно 6 байт (48 бит), всегда 3 группы по 4 hex-символа. Уникален в пределах зоны. Часто берётся из MAC или IP
N-SEL Network Selector. На маршрутизаторе всегда 00. Ненулевые значения — для ES (end-systems, не роутеры)

Сценарий: Один роутер Arista vEOS, Loopback0 = 10.0.0.1/32, зона 1.
Конвертируем IP в System ID: 10.0.0.1 → дополняем до 12 цифр → 010.000.000.001 → 0100.0000.0001
49 . 0001 . 0100 . 0000 . 0001 . 00
AFI 49 Стандартное значение IS-IS
Area ID 0001 Зона 1
System ID 0100.0000.0001 из IP 10.0.0.1 → 010.000.000.001
N-SEL 00 Роутер (всегда 00)
! Arista vEOS — роутер R1 router isis UNDERLAY net 49.0001.0100.0000.0001.00 is-type level-2 ! interface Loopback0 ip address 10.0.0.1/32 isis enable UNDERLAY
Сценарий: System ID из MAC-адреса интерфейса Management.
MAC: 00:1c:73:ab:cd:ef → убираем : → 001c.73ab.cdef
49 . 0002 . 001c . 73ab . cdef . 00
AFI 49 Стандартное значение IS-IS
Area ID 0002 Зона 2 — другая зона
System ID 001c.73ab.cdef из MAC 00:1c:73:ab:cd:ef
N-SEL 00 Роутер (всегда 00)
! Arista vEOS — System ID из MAC router isis UNDERLAY net 49.0002.001c.73ab.cdef.00 is-type level-2 address-family ipv4 unicast
Сценарий: L1/L2 роутер, стоящий на границе двух зон. Роутер имеет два NET — по одному на зону. Area ID может быть длиннее (до 13 байт).
Зона 1: 49 . 0001 . 0a0a.0a0a.000a . 00
Зона 2: 49 . 0002 . 0a0a.0a0a.000a . 00
! L1/L2 border router — два NET (System ID одинаковый!) router isis UNDERLAY net 49.0001.0a0a.0a0a.000a.00 net 49.0002.0a0a.0a0a.000a.00 is-type level-1-2 ! System ID должен совпадать в обоих NET!
Сценарий: Фабрика Spine-Leaf на Arista. Spine = зона 49.0001, Leaf = та же зона. System ID из Loopback IP каждого устройства.
! ── SPINE-1 (Loopback0: 10.255.0.1) ────────────── router isis EVPN_UNDERLAY net 49.0001.1025.5000.0001.00 is-type level-2 router-id ipv4 10.255.0.1 ! ── SPINE-2 (Loopback0: 10.255.0.2) ────────────── router isis EVPN_UNDERLAY net 49.0001.1025.5000.0002.00 is-type level-2 ! ── LEAF-1 (Loopback0: 10.255.1.1) ────────────── router isis EVPN_UNDERLAY net 49.0001.1025.5001.0001.00 is-type level-2 ! ── LEAF-2 (Loopback0: 10.255.1.2) ────────────── router isis EVPN_UNDERLAY net 49.0001.1025.5001.0002.00 is-type level-2
